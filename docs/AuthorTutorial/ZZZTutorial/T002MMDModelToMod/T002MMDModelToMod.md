# 将MMD模型转为Mod流程

因为新手做Mod时，大部分都不会自己建模，所以这里用获取最方便的MMD模型来举例

可以在模之屋找个模型跟练：https://www.aplaybox.com/

我们在实际制作的过程中，用什么模型都是可以的，所以制作Mod的前提是你熟练掌握Blender的模型处理技巧

在大多数情况下，SSMT与TheHerta在整个Mod制作过程中，只能占用你1%不到的时间，在你熟练使用SSMT流程之后，99%的时间都会花在Blender中
