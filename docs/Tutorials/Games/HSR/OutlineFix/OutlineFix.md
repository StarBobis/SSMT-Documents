# 📏 崩铁轮廓线修复

## 🛠️ 解决方案

### 方法一：生成时修复

在 Catter 中勾选 **【向量归一化法线存入TANGENT(全局)】** 后生成的 Mod 的轮廓线即可得到修复。

![alt text](image.png)

### 方法二：手动修复

也可以 **右键模型** => **3Dmigoto** => **使用向量相加归一化算法重计算TANGENT**，来精确控制每个物体是否需要修复轮廓线。

![alt text](image-1.png)

> ⚠️ **注意**：此方法只能 **99%** 修复身体轮廓线，角色头发的轮廓线只能得到 **60%** 近似修复。
