# 崩铁无法正常提取模型的BUG

此问题是由于SRMI更新后的Custom Shader替换方式导致的，在替换后并未还原Buffer

所以如果仅仅是按F6关闭Mod的话，也会导致我们F8 Dump出来的Buffer文件是错误的

也就导致了提取出来的模型不正确或者根本无法提取的问题

# 解决方式

解决方法是在F8 Dump之前，先 `DISABLED` 掉所有的Mod，也就是相当于清空Mods文件夹的内容

然后游戏里F10刷新，切换一下场景让模型重新加载，再Dump才能正常提取

最优的解决方案是在SSMT里新建一个游戏配置专门用来做Mod或者玩Mod，把做Mod和玩Mod使用的加载器分开，这样就不会被干扰了。
