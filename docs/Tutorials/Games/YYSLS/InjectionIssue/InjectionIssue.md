# ⚠️ 多个游戏主程序问题

**YYSLS** 有俩路径，一个是 `Win64rh`，一个是 `Win64r`。
游戏启动时，会随机启动其中一个。

-   如果你填写 `win64rh` 的话，有时候它启动的是 `win64r` 的。
-   如果你填写 `win64r` 的话，有时候它启动的是 `win64rh` 的。

> 🚫 **误导性警告**：但是即便进程路径选错了，这里注入器也会显示注入成功，具有迷惑误导性。

![alt text](image.png)

## 🛠️ 解决方案

此时最好是先启动一遍 YYSLS 主程序，然后在任务管理器中查找进程名称，确定启动的到底是哪个，然后再去 3Dmigoto 配置中填写正确的进程路径。

![alt text](image-1.png)

> 💡 **小贴士**：当然有时候其实你已经注入成功了，但是由于绿字界面默认是不显示的，此时可以按小键盘 `0` 开关绿字显示。
