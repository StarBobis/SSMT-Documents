# 🛡️ 插件说明

**SSMT-Protect** 插件是用于 **3Dmigoto Mod** 的锁机器码和 Mod 加密的资产保护工具。

## ✨ 主要功能

- 🔒 **防倒卖/防搬运**：有效防止 Mod 被非法倒卖或搬运。
- 🤝 **防合购**：防止多人合购一份 Mod。
- 🧩 **防逆向提取**：防止 Mod 模型被逆向提取。
- 💎 **资产化**：把你的作品垄断并变成资产。
- ♾️ **永久使用**：一次激活即可永久使用。
- 🔄 **持续更新**：定期更新防破解机制。
