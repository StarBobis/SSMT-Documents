# ⚔️ 3Dmigoto-Sword-Lv5 插件

**3Dmigoto-Sword-Lv5** 是一款专业的用于将 **3Dmigoto Mod** 里的模型提取出来的工具，它的工作原理是全自动分析 `ini` 文件中的语法，并构建抽象的逻辑分析链进行语法分析，从 `ini` 内容中识别出 Buffer 文件可能组合成的模型文件。现在作为 **3Dmigoto-Sword-Lv6** 的插件使用。

> 💰 **获取方式**：
> 可以在下方链接中赞助获取：
>
> [https://afdian.com/item/ec74ee782b2f11efb5a052540025c377](https://afdian.com/item/ec74ee782b2f11efb5a052540025c377)

工具的 GUI 界面是开源的，名为 **3Dmigoto-Sword-Lv6**，开源地址为 [https://github.com/StarBobis/3Dmigoto-Sword-Lv6](https://github.com/StarBobis/3Dmigoto-Sword-Lv6)

其天生具备开源免费的 **【手动逆向】** 功能，此篇文档主要介绍其闭源插件 **3Dmigoto-Sword-Lv5** 中更加方便的一键逆向。

## ✨ 功能强大

- 🚀 **一键还原**：无需开启游戏，只要获取到 Mod 文件，就能一键分析 `ini` 的方式把 Mod 里的模型逆向还原为 Mod 文件。
- 🧠 **智能解析**：超复杂 `ini` 解析，模拟 3Dmigoto 解析逻辑。
- ⚡ **极速内核**：C++ 编写的内核，运行速度超快，3秒内还原模型。
- 🖥️ **流畅界面**：操作界面由 WinUI3 编写，极速响应无卡顿。
- 🛡️ **权益保障**：激活机制保障用户权益，防止工具被恶意倒卖损坏用户权益。
- 📦 **便捷使用**：设计简洁解压即用，下载和更新都很方便。
- 🔍 **全面逆向**：可以把所有可能的数据类型都逆向出来，确保逆向结果正确。
- 💯 **高兼容性**：基本上 99% 的 Mod 都能逆向，如果配合手动修改 `ini` 以及手动逆向，则所有支持的类型的开源格式 Mod 都能逆向。
