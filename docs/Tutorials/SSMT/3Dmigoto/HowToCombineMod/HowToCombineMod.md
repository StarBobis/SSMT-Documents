# 🔗 如何合并多个 Mod 为一个按键切换 Mod

## 📝 概述

使用 **SSMT** 制作的 **Mod** 建议使用基于 **namespace** 的合并脚本来制作多合一 **Mod**。

## 🛠️ 推荐工具

- **基于 namespace 的合并脚本**: [https://gamebanana.com/tools/15804](https://gamebanana.com/tools/15804)
- **另一种合并脚本**: [https://gamebanana.com/tools/15681](https://gamebanana.com/tools/15681)

> ⚠️ **注意**：其它合并脚本对于 **SSMT** 的格式可能存在不兼容的情况，请优先使用上述推荐工具。
