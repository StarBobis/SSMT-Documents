# 🖼️ 模型单面贴图问题

## 🚫 问题描述

有些 **MMD** 模型的头发、裙子只有一个面，正面显示贴图，背面就显示纯黑色了。

## 🛠️ 解决方案

### 方法一：使用实体化修改器

我们可以使用 **实体化修改器** (**Solidify**) 来解决：

1. 距离选 `0.01m`。
2. 取消勾选 `Fill`。
3. **Normal** 选 `Flip` 即可生成出一个完全相反的面来。
4. 记得做之前先分割出来。

### 方法二：使用 TEXCOORD1 或以上控制内面贴图

有些游戏比如 **GI** 的某些模型，使用 **TEXCOORD1** 或更高的 **TEXCOORD** 来控制模型衣服里面显示的内容。

> 💡 **提示**：根据模型的具体情况选择合适的方法，确保贴图在正面和背面都能正确显示。
