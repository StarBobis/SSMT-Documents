# 🚨 提取模型时找不到数据类型？别慌！这里有解决方案 💡

嘿，Mod 爱好者们！在 SSMT3 中提取模型时，如果遇到 **找不到数据类型** 的提示，别担心，这篇指南会帮你一步步解决！🔧

## 🔍 初步排查：运行日志是你的好帮手

当提取失败弹出 **找不到数据类型** 的提示时，首先别急着求助！按照以下步骤逐个排查：

- 📋 **检查运行日志**：运行日志中的报错提示往往能揭示真相。有时候问题不是真的缺少数据类型，而是 **游戏预设** 设置错误导致的。
- 🔍 **逐个排查**：仔细阅读日志，找出可能的错误原因。记住，日志是调试的第一步！

如果自己排查不出问题，那可能真的是缺少数据类型。别慌，继续往下看！

## 📦 F8 Dump 出来的 FrameAnalysis 文件夹太大？压缩攻略！

F8 Dump 出来的 **FrameAnalysis 文件夹** 特别大？别担心，这里有几个技巧帮你减小体积和提升速度：

- 🎯 **减少 IB 总数**：Dump 时，让当前的绿字中 **IB 的总数** 尽可能少。越少，Dump 出来的文件大小越小，速度越快！
- 🔗 **开启 Symlink 特性**：这个特性能显著减少 Dump 文件大小，并提升 Dump 速度。强烈推荐开启！

![alt text](image.png)

- 🔄 **还原绿字界面**：Dump 之前，务必按下 **小键盘 + 号** 来还原绿字界面到初始状态。
- 👀 **视野优化**：视野内渲染的物体越少，**IB 的数量** 就越少。在大部分游戏中，抬头看天空，或尽可能让视野里只有这一个物体，来让 IB 值尽可能减小。
- 🎛️ **指定 IB 列表 Dump**：如果你会用这个功能，就开启它然后 Dump 发给我。如果不会用，就 Dump 完整的就好。
- 📏 **文件太大怎么办**：如果 Dump 文件实在太大，直接告诉我这个东西在游戏里的什么地方，叫什么名字。我可以直接去游戏测试（虽然我不玩游戏，但可以试试）。大部分情况还是需要你发我文件。
