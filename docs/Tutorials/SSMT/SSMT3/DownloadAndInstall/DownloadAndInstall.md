# 🚀 SSMT3 下载与安装指南

欢迎来到 **SSMT3** 的世界！这款基于 **3Dmigoto** 的模型提取与 Mod 制作工具箱，将为你带来无限创意可能。🎨

## ⚙️ 安装环境要求

在开始安装前，请确保你的系统满足以下条件，以保证 **SSMT3** 运行顺畅：

- 💻 **系统兼容**：**SSMT3** 仅能在正常 PC 的 **Windows 11** 系统中正常使用。在其他系统或虚拟机上使用可能会有 BUG。
- 🔧 **运行时环境**：需要安装 **.NET 8.0** 运行时环境，请自行安装好。
- 💾 **存储建议**：强烈建议安装到 **固态硬盘 (SSD)** 上，并预留至少 **200G** 空闲空间。最好放在桌面解压即用。
- 🚫 **禁止位置**：
  - 不能放在移动硬盘、U盘等无权限磁盘下。
  - 不能放在 **机械硬盘 (HDD)** 上，否则部分操作可能会十分缓慢，例如 F8 Dump 时如果使用默认位置下的 3Dmigoto 就会由于机械硬盘速度缓慢而 Dump 缓慢。
  - 不能放在空间较小的硬盘上，因为 F8 Dump 的内容比较大，如果用的是默认位置下的 3Dmigoto 就可能会导致频繁 F8 Dump 的内容把硬盘撑满。
- 🛡️ **安全设置**：请先关闭微软的 **Smart Screen**，防止无法启动注入器。设置路径：设置 → 隐私和安全性 → Windows 安全中心 → 应用和浏览器控制 → 基于声誉的保护设置。

> ⚠️ **重要提醒**：为了排除一切可能的风险，确保安装顺利，建议不要放在移动磁盘、U盘等低权限或无权限位置，防止 SSMT 无法正常使用。
