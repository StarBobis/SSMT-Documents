# SSMT3 下载与安装

SSMT3是一款基于3Dmigoto的模型提取与Mod制作工具箱

# 安装环境
- SSMT3仅能在正常PC的Windows11系统中正常使用，在其它系统以及虚拟机上使用可能会有BUG。
- SSMT3的运行需要安装.NET 8.0运行时环境，请自行安装好。
- SSMT建议安装到固态硬盘上，并预留至少200G空闲空间，最好是放桌面解压即用。(其他位置也可以，但少数特殊情况可能会导致难以排查的权限问题，尽管SSMT本体不需要管理员权限即可运行，但为了排除一切可能的风险，确保安装顺利，建议不要放在移动磁盘、U盘等低权限或无权限位置，防止SSMT无法正常使用)
- 不能放在移动硬盘、U盘等无权限磁盘下。
- 不能放在机械硬盘上，否则部分操作可能会十分缓慢，例如F8 Dump时如果使用默认位置下的3Dmigoto就会由于机械硬盘速度缓慢而Dump缓慢。
- 不能放在空间较小的硬盘上，因为F8 Dump的内容比较大，如果用的是默认位置下的3Dmigoto就可能会导致频繁F8 dump的内容把硬盘撑满。
- 请先关闭微软的Smart Screen，防止无法启动注入器，设置→隐私和安全性→Windows 安全中心→应用和浏览器控制→基于声誉的保护设置
