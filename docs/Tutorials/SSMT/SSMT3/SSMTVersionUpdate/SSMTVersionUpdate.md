# 🔄 SSMT 版本更新

## 📝 自动更新

切换到 SSMT 的设置页面，右下角按钮可以 **自动更新版本**：

![alt text](image.png)

## 🛠️ 手动更新

如果手动更新的话，可以去 **GitHub** 或者我发的网盘链接或者群文件等等下载 SSMT 的压缩包，解压到桌面就能直接使用。

## ❓ 常见问题

### 🔧 自动更新按钮报错怎么办？

一般情况报错可能是由于你的网络无法连接到 **GitHub** 导致的，因为自动更新就是自动去 GitHub 下载并更新最新版。

> 💡 **建议**：你需要开加速器或者梯子、魔法等等，确保你能联通 GitHub 再点击自动更新。

### 📁 更新需要覆盖旧版本吗？

**不需要**，解压出来就能直接使用。

### 📥 GitHub 这么多选项下载哪一个？

![alt text](image-1.png)

-   **优先建议**：下载 `SSMT-win-Portable.zip`，这个解压到桌面就能直接用。
-   **安装包**：如果你喜欢安装包，也可以下载 `SSMT-win-Setup.exe`，但是安装包是只能一键安装到 C 盘的，无法自定义路径。
