# 🦴 如何删除骨骼约束

如图所示，导入 MMD 模型后，绿色的大腿根部骨骼约束导致腿部不能按照特定角度旋转。

这是因为 Blender 中 **黄色** 和 **绿色** 的骨骼都是加了约束的。

![alt text](image.png)

## 🛠️ 解决方法

1. 点击这个 **绿色的约束骨骼**。
2. 进入右侧属性面板的 **骨骼约束栏**（图标类似骨头加链条）。

![alt text](image-1.png)

3. 把右边的约束 **全部删掉**，有几个删几个。

![alt text](image-2.png)

4. 删掉约束的骨骼会变成 **灰色**。我们删掉了旋转约束，这样就可以自由旋转了！

![alt text](image-3.png)
